# C++_Calculator_Project

## About

This is a simple calculator built in C++. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Avoids division by zero 

## What I Learned

- **C++ Basics**: I learned how to use basic tools like iostream to talk to users.

- **User Input**: I practiced using cin and cout to read numbers and symbols from the keyboard.

- **Variables**: I learned how to store decimals using the double data type.

- **Control Flow**: I mastered the if, if else, else and switch-case statements to run different code based on the operator chosen and certain conditions.

- **Error Handling**: I learned to protect my code from crashing by stopping division by zero.

- **Syntax Precision**: I realized the importance of matching all curly braces {} to avoid code errors.
